# To-do-List
simple to-do list
This is a simple Python command-line Todo List app that allows users to add, view, complete, delete, and clear tasks. The program saves all tasks in a my_todos.json file so the list stays saved even after closing the program. Users can mark tasks as done, remove completed tasks, and manage their list through an easy menu system. This project helps practice Python concepts such as functions, lists, dictionaries, loops, file handling, JSON, and user input.
